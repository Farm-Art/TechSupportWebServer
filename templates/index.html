Набор фигур ГОСТ-19.701-70 для Microsoft Visio (текущая версия: 6)

Установив Visio, я не обнаружил там пакета графики, соответствующего ГОСТу, приложенному к заданию. Это может быть связано с тем, что я ставил англоязычную версию программы, а может быть и не связано. В любом случае, пакет всё равно был нужен, поэтому я его нарисовал сам. Пакет содержит:

  1. Нод "Терминатор" (Начало/Конец работы программы)
  2. Нод "Процедура"
  3. Нод "Ввод/Вывод"
  4. Нод "Внешняя процедура"
  5. Нод "Решение" (if/switch...case)
  6. Ноды начала и конца цикла (for)
  7. Нод "Подготовка данных" (for в один блок)
  8. Нод "Соединитель"
  9. Нод "Комментарий"
  10. Ноды "Динамический Соединитель.С" и "Динамический Соединитель.Б"
  11. Шрифт "ГОСТ type-A курсив" в формате truetype (.ttf)

Все ноды предоставлены в двух вариациях: Цветная (автоматически адаптируется под текущую тему), и ЧБ (не зависит от темы, идеально для печати). К пакету приложен ГОСТ-шрифт, рекоммендуемый Вершининым для установки и последующего использования в документации. При желании, можно скачать предыдущую версию пакета без шрифта, но это надо быть мазохистом. Ноды вида "Динамический Соединитель" представлены лишь в цветном экземпляре, что связано с патологической неспособностью Microsoft делать простой и понятный разработчику каких-либо доп. ресурсов интерфейс особенностями программного обеспечения. Каждый нод обладает как минимум 4-мя точками соединения: сверху, снизу, слева, справа. Каким-то нодам это положено "по профессии", у каких-то это просто упрощает процесс прикрепления комментария.

Инструкция по установке

Для добавления пакета в Visio, перетащите файл пакета (.vssx) в программу и откройте появившуюся вкладку. Если вы уже читали инструкцию из этой секции, забудьте её напрочь, ибо Microsoft показали нам корпоративный средний палец. Вот краткий гайд для тех, кому плевать на нюансы:

  1. Создайте одну линию с помощью инструмента
  2. Откройте Допольнительные фигуры -> Набор элементов документа
  3. Найдите там объект "Линия связи", кликните дважды
  4. Измените его, как душа пожелает
  5. Закройте окно и подтвердите изменение шаблона

После сохранения все существующие и последующие линии обновятся и будут совпадать по цвету с темой и по формату с Линией связи. Чёрно-белая тема включается во вкладке "Конструктор" (именно так они перевели "Design", лол) -> "Без темы".

Теперь информация для любопытных: Microsoft ищет не просто элемент "Линия связи" в наборе документа, а записывает ID конкретного элемента "Линия связи", созданного автоматически Visio. Переименовать элемент из набора документа в линию связи нельзя; можно только перетащить элемент с таким названием, и даже в таком случае программа просто создаст новый элемент с числовым постфиксом, который затем запоминается, как "Официальный" соединитель. Единственный способ как-то изменить линии - изменить официально созданный элемент "Линия связи", иначе Visio просто его проигнорит и создаст свой. Браво, Microsoft, апплодирую стоя.
